# Prime Number Checker in C

A simple C program that checks whether a given number is prime or not.

## Features

* Takes input from the user
* Checks if the number is prime
* Uses loops and conditional statements
* Beginner-friendly implementation

## Concepts Used

* Variables
* User Input
* for Loop
* if-else Statements
* break Statement
* Prime Number Logic

## Example

Input:
13

Output:
The number 13 is prime

Input:
12

Output:
The number 12 is not prime
